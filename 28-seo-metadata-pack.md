# SEO and Social Metadata Pack (All Channels)

This document defines the SEO, metadata, and social sharing assets for the AI-native website. It must stay factual, non-sales, and consistent with the AI interface tone.

## Goals
- Ensure clean previews across all social platforms.
- Provide consistent brand positioning in search results.
- Avoid hype; remain precise and credible.

---

## Global Defaults (Site-Wide)
**Title Format**
- "[Primary Page Title] | [Brand Name]"

**Default Meta Description**
- "An AI-only intelligence partner designing systems for automation, reasoning, vision, and voice."

**Robots**
- index, follow

**Canonical**
- Use full canonical URLs per page.

---

## Social Preview Assets
**Primary OG Image**
- Size: 1200 x 630
- Style: dark base, subtle neural grid, minimal text.
- Text: "Enter the Intelligence Layer"

**Secondary OG Image (for technical pages)**
- Size: 1200 x 630

- Type: summary_large_image

## Global Open Graph Tags (Defaults)
- og:type: website
- og:site_name: [Brand Name]
- og:title: [Page Title]
- og:description: [Page Description]
- og:image: [Primary OG Image URL]
- og:url: [Canonical URL]

## Global Twitter/X Tags (Defaults)
- twitter:card: summary_large_image
- twitter:site: [@brandhandle]
- twitter:title: [Page Title]
- twitter:description: [Page Description]
- twitter:image: [Primary OG Image URL]
