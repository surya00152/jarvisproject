# Page Prompt: Security and Compliance

This page is a technical trust center. It should be precise, grounded, and enterprise-ready, without making unverified certification claims.

## Page Intent
- Explain security architecture and operational controls.
- Clarify deployment options and data boundaries.
- Provide compliance alignment without overstatement.

## CTA Strategy (page level)
- Primary CTA: "Talk to the AI Architect" (voice).
- Secondary CTA: "Privacy and Ethics".

---

## Section 1: Security Posture Hero
**Purpose**
- Set a calm, confident security tone.

**Content Intent**
- Headline: "Security is part of the intelligence."
- Subtext: "Every layer is designed for control, auditability, and safety."

**Design and Animation Logic**
- Minimal grid with locked-node highlights.

---

## Section 2: Security Architecture Overview
**Purpose**
- Provide a clear, system-level view of security layers.

**Content Intent**

- Audit logging and monitoring.

**Design and Animation Logic**
